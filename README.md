## ✨ Diferenciais do LodiMusic

Por que usar o **LodiMusic Ultimate**? Diferente dos bots convencionais, ele utiliza o motor **Lavalink**, garantindo:

* 🚀 **Zero Lag:** Processamento de áudio externo ao bot.
* 🎧 **Filtros DSP:** Bass Boost, Nightcore, Slowmo e modo Karaoke.
* 📜 **Lyrics Integrado:** Letras em tempo real via Genius API.
* 🖼️ **Visual Premium:** Embeds customizadas com a arte do álbum.
* 🔗 **Multi-Plataforma:** Suporte para YouTube, Spotify e SoundCloud.

## 🛠️ Comandos Principais

| Comando | Descrição |
| :--- | :--- |
| `!play [nome/link]` | Toca uma música ou playlist. |
| `!filter [tipo]` | Aplica filtros (bassboost, nightcore, slow, karaoke). |
| `!lyrics` | Busca a letra da música que está tocando agora. |
| `!queue` | Exibe a fila de reprodução atual. |
| `!skip` | Pula para a próxima faixa da fila. |
| `!stop` | Para a música e desconecta o bot. |
